Applai Grants — Intelligent Grant Matching & Proposal Generator
Deliverables: Containerized AI microservice (FastAPI) with /match and /generate endpoints; FAISS embeddings; PDF parsing pipeline; matching logic; proposal formatter; error samples & retry logic.
What will be built: A lightweight, modular AI service that connects with Applai's platform to provide smarter grant matching using semantic vector search and tailored proposal generation from user pitch decks and profiles.
  • Smarter Grant Matching: semantic search + tag blending
  • Proposal Generation: section-by-section proposal drafts using open models
  • PDF Integration & profile enrichment
Tech stack: FastAPI (Python), PyPDF2 + OCR fallback, SentenceTransformers embeddings, FAISS (local), Mistral 7B (4-bit quantized), Docker.

Sunrisal — Machine Learning for Rural Electrification
Role: Designing ML-powered customer profiling and PayGo payment plan generation to expand access to solar home systems.
Sunrisal combines solar home systems with an ethical ML profiling app to propose customized PayGo payment plans for Nano and MSMEs based on socio-economic factors (income, gender, location). After a tailored plan and downpayment, customers deploy the solar kit to power their business.

Suggested tech stack: FastAPI backend, PostgreSQL (or Supabase R2 for storage), Redis for caching, PyTorch/Transformers for lightweight models, Celery for background jobs, S3-compatible storage (R2), Docker + Kubernetes for deployment, USSD integration (Paga) for payments.
